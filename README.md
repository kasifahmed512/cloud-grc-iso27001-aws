Cloud GRC Implementation (ISO 27001) on AWS

1.	Overview
   
This project demonstrates implementation of an ISO 27001-based ISMS using AWS services.


2.	Architecture
   
CloudTrail → CloudWatch / EventBridge → Lambda → SNS


3.	Key Features
   
•	IAM security (MFA, RBAC, least privilege)
•	S3 encryption (SSE-KMS)
•	CloudTrail logging and monitoring
•	AWS Config compliance tracking
•	Automated remediation using Lambda
•	Real-time alerts via SNS

4.	Outcome
   
•	Reduced MTTR to <2 minutes
•	Achieved high compliance visibility
•	Built audit-ready cloud environment
